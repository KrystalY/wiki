# CSS

## Set Default
  > 초기화 설정
  > initial? inherit?

  ```css
  width: auto;
  background-color: transparent;
  mini-width: inherit || auto;
  ```
  각 태그에 맞는 초기화를 설정하자
  * initial 사용시, ie 에러등 많이 

  #### # 참고
  - [MDN- 상속](https://developer.mozilla.org/ko/docs/Web/CSS/inheritance)
  - [MDN - inherit](https://developer.mozilla.org/ko/docs/Web/CSS/inherit)
