# CSS

## Set Defaault
  > 초기화 설정
  > initial? inherit?

  ```css
  width: auto;
  background-color: transparent;
  mini-width: inherit || auto;
  ```
  각 태그에 맞는 초기화를 설정하자
  * initial 사용시, ie 에러등 많이
